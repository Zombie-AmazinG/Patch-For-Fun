# Patch By Zombie AmazinG
### This patch has
- Timer
- Round Timer
- Box Hit
- Seconds Per Horde (only on shi no numa and der riese)
- Dog Tracker (only on der riese)
- Trap Timer (except on nacht)
- Zombie Counter
- Health Bar
